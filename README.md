# Gruenstifter × Machine.Machine

Slide-based web presentation explaining how Machine.Machine AI can accelerate Artenschutzgutachten workflows for [Gruenstifter](https://gruenstifter.com).

**Live**: [gst.machinemachine.ai](https://gst.machinemachine.ai)

## Stack

- Single `index.html` — self-contained, no build step
- CSS scroll-snap slide system
- Dark theme matching pitch.machinemachine.ai
- Mobile-friendly
- Google Fonts (Inter)

## Deploy

Deployed via Coolify webhook on push to `main`.
